Metodologia de Viabilidade

Como o sistema avalia cada CP

Objetivo: Avaliar se uma campanha de IR (Venda Direta do Futuro) é financeiramente viável para um dado CP, cruzando histórico de resultados reais com o tamanho da audiência digital e a verba alocada.
Pilar 1 — Performance Histórica
Calcula o CPCadastro real do CP (Investimento ÷ Cadastros realizados) e compara com a mediana da base completa de todos os CPs com histórico.
🟢 Eficiente
CPCad ≤ 90% da mediana
🟡 Na média
CPCad entre 90%–140%
🔴 Acima
CPCad > 140% da mediana
Pilar 2 — Saturação de Audiência
Mede o investimento por habitante digital (Verba ÷ Pop. Digital Estimada). A população digital é obtida via API do IBGE para cada município do CP, multiplicada pelo fator de penetração configurável (padrão 84%).
🟢 Baixa
< R$0,50/hab
🟡 Moderada
R$0,50–R$1,20/hab
🔴 Alta
> R$1,20/hab
Pilar 3 — Confiabilidade dos Dados
Avalia o volume de cadastros históricos para saber quão confiável é a estimativa. CPs com poucos cadastros históricos têm alta variância — o CPCadastro pode não ser representativo.
⚪ Sem dados
0 cadastros
🟡 Baixo
< 10 cadastros
🟡 Razoável
10–49
🟢 Sólido
≥ 50 cadastros
Decisão final — O semáforo combina os 3 pilares:
🟢 Viável — Performance boa + saturação baixa/moderada
🟡 Atenção — 1 pilar em risco ou dados insuficientes
🔴 Inviável — Performance ruim e saturação alta, ou saturação crítica sem histórico
⚪ Sem dados — Nenhum histórico real disponível para o CP
